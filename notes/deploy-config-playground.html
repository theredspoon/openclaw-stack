${VAR} — .env + stack.yml resolution

At npm run pre-deploy, ${VAR} and ${VAR:-default} references in stack.yml are resolved from .env values. Supports defaults via :- syntax. Comments are preserved.

              Stage 3
              
$VAR — Runtime envsubst at container startup

            

            

              openclaw.jsonc is parsed to JSON (comments stripped) during pre-deploy.
              At container startup, entrypoint.sh runs envsubst on the
              JSON with a whitelist of allowed variables. Docker env vars (set in compose) provide
              the values.
